# Lab 05 (b) – HCI Principles Demo

## Overview
This demo shows Norman’s principles (affordance, signifiers, feedback, mapping, constraints) using a button interface with colored visual cues.

## Project Structure

Lab05b/
├── index.html   # Structure
├── style.css    # Style (colors, borders, effects)
└── script.js    # Behavior (click logic, feedback, constraints)


## Outcome
- **Affordance:** Blue border shows clickability.  
- **Signifiers:** Label and purple disabled state guide user.  
- **Feedback:** Yellow flash + status text confirm action.  
- **Mapping:** Green border links action to status area.  
- **Constraints:** Red border + disabled button prevent errors.
