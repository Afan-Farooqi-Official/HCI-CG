# Lab 05 (a) – Separation of Concerns

## Overview
This lab refactors the HTML button example to separate **structure (HTML)**, **style (CSS)**, and **behavior (JavaScript)**.

## Project Structure

Lab05a/
├── index.html   # Structure
├── style.css    # Style
└── script.js    # Behavior


## How It Works
- **index.html** defines the button and status area.  
- **style.css** handles design and hover/disabled effects.  
- **script.js** manages click logic, feedback, and constraints.  

## Outcome
The code is cleaner, modular, and easier to maintain.
